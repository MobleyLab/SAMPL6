TechnicalDetails:
All the simulations were performed on two nodes of the Bridges system, equipped with 2 Intel Haswell (E5-2695 v3) CPUs 14 cores/CPU, 2.3 - 3.3 GHz. Single equilibration runs were peformed on the first configuration file of each host-guest pair and are not reflected in the cost of each system (see methods section). Total wall time is estimated based on the average wall time for the system.

This work used the Extreme Science and Engineering Discovery Environment (XSEDE), which is supported by National Science Foundation grant number OCI-1053575.  Specifically, it used the Bridges system, which is supported by NSF award number ACI-1445606, at the Pittsburgh Supercomputing Center (PSC).

Method:

Fully  atomistic,  expanded  ensemble  simulations  were  run for  each  system  using  GROMACS  with  time  steps  of 0.002 ps and the md-vv integrator. V-rescale  temperature  coupling and Berendsen pressure  coupling were  both  used  and  bonds  to  hydrogen  atoms  were constrained  using  the  SHAKE  algorithm.  Metropolized  Gibbs  Monte  Carlo  moves between  all intermediate  ligand  coupling states were  performed  every 100 time steps based on weights calculated with the Wang–Landau  algorithm.  The  metropolized  Gibbs  move  in state space proposes jumps to all states except the current state, with a rejection step to satisfy detailed balance. For all systems an absolute value of the initial Wang–Landau incrementor was set to 2 kBT. Weights were updated at each step, and the increment amount was reduced by a factor of 0.8 each time a flat histogram was observed, meaning that the ratio between the least visited and most  visited  states  since  the  last  change  in  the  weight increment  was  less  than  0.7.  The  process  of  updating  the weights was halted when the incrementing amount fell below 0.001 kBT. Simulations where the weights were updating are termed the "equilibration" stage. Equilibration was only ran on a single starting conformation for each
host-guest pair. A sample .mdp file can be found below. The weight of the fully coupled state is normalized to zero, meaning that the weight of the uncoupled state corresponds to the free energy of the process.

The  last  stage  of  the  simulation,  during  which  period the WL  weights  were  no  longer  updated,  was  termed  the "production"  stage  since  it  was  the  only  part  of  the  trajectory used to calculate the final free energy change. Once the Wang–Landau incrementor reached a value of 0.001 the simulation was stopped, MBAR was ran on simulation data obtained while the Wang–Landau incrementor was between values of 0.01 and 0.001 kBT, and the results were used to run production simulations for each starting conformation of a host-guest pair. 

In this submission the time of each equilibration stage was included in the total wall-time and energy evaluations. Because a single pair of equilibration simulations were run for each system the time was divided equally across the five replicates. Instead, only production data was used to determine estimates of free energy difference for each system. An equal number of time steps were allocated to production simulations of complex and solvent systems for each free energy estimate. Reported values were obtained by running MBAR on production simulation data. If sampling was insufficient to run MBAR a zero is reported.
